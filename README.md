# Maishapay Gateway

Centralized API gateway for Maishapay payments. Implements a Node.js + TypeScript Express server and helpers to interact with Maishapay and Supabase.

Getting started

1. Copy `.env.example` to `.env` and set values.
2. Install dependencies: `npm install`.
3. Develop: `npm run dev` (configure `dev` script if needed).
4. Build: `npm run build` and start: `npm start`.

Files added:
- `src/types/maishapay.types.ts`
- `src/services/maishapay.service.ts`
- `src/services/database.service.ts`
- `src/index.ts`
